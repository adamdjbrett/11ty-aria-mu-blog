Welcome to your **Eleventy 3.1.2** blog styled with [MU CSS](https://unpkg.com/@bafs/mu@0.3/mu.min.css) and using Arial Black for headings and Arial for body text.

- Edit or add posts in `src/posts/`.
- Start the dev server with `npm run dev`.
- Build a production site with `npm run build`.
